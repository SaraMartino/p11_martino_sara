Per importare correttamente il progetto in Eclipse, scaricare la cartella src sul proprio computer
e successivamente eseguire le seguenti operazioni:

   1. 	File -> Import
   2. 	Espandere Gradle e selezionare Gradle Project (premere Next)
   3. 	Saltare la schermata di Welcome del Gradle (nel caso in cui dovesse comparire)
   4. 	Selezionare la directory p11_GestionePrenotazioneCinemaMultisala all’interno di src come "Project root directory"
	e premere Finish
   5. 	Nel caso in cui dovesse comparire una finestra di avviso (Overwrite existing Eclipse project descriptor?)
	premere Overwrite
